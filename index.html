Panduan HTML: Koding Seru Tanpa Stress!

Hai, Sobat Koder! Selamat datang di dunia HTML! Di sini, kita bakal belajar cara bikin halaman web sendiri. Siap-siap, ya! Kita akan ketawa sambil belajar!

Apa Itu HTML?

HTML itu kayak baju yang kamu pakai. Tanpa baju, kamu bakal jadi pusat perhatian, tapi bukan perhatian yang baik! HTML ini adalah struktur dasar dari sebuah halaman web. Tanpa HTML, website kita bakalan telanjang, Sob! Jadi, mari kita pakaikan HTML yang kece ini ke website kita biar tampilannya cakep.

HTML adalah singkatan dari Hypertext Markup Language, yaitu bahasa markup standar untuk membuat halaman web. Ini adalah pondasi dari semua halaman web, ibarat pondasi bangunan sebelum kita menghiasnya dengan CSS dan JavaScript. Yuk, kita lanjut belajar bagaimana cara pakai HTML ini!

Struktur Dasar HTML

Sebelum kita beraksi, yuk lihat struktur dasar HTML. Seperti bikin rumah, kita harus mulai dari fondasinya:

<!DOCTYPE html>
<html>
<head>
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Selamat Datang di Halaman Ini!</h1>
    <p>Ini adalah paragraf pertama saya. Ayo ngopi!</p>
</body>
</html>

Penjelasan:

<!DOCTYPE html> Ini semacam ucapan selamat datang, bilang ke browser, "Hey, ini HTML, loh!"

<html> Ini adalah rumah untuk semua konten kita, di sini semua tag HTML lainnya akan tinggal.

<head> Bagian ini untuk informasi meta, judul, dan link ke file CSS atau JavaScript. Browser butuh ini buat ngertiin halaman kita.

<body> Di sini tempatnya semua konten kita akan ditampilkan, seperti teks, gambar, dan lainnya.

Judul dan Sub Judul

Sekarang kita belajar bikin judul dan sub judul. Judul itu penting banget, karena itu yang pertama dilihat oleh pengunjung website kamu.

Contoh Judul dan Sub Judul:

<h1>Acara Koding Keren</h1>
<h2>Sub Judul: Belajar HTML Tanpa Stress</h2>

Penjelasan:

<h1> adalah judul utama, digunakan sekali dalam satu halaman. Kalau sub judul, gunakan <h2> atau lebih rendah.

Paragraf

Paragraf adalah cara kita bercerita atau menyampaikan informasi dalam website. Kita bisa buat sebanyak mungkin paragraf untuk menjelaskan berbagai hal.

Contoh Paragraf:

<p>Ini adalah contoh paragraf sederhana.</p>

Penjelasan:

<p> adalah tag untuk membuat paragraf. Segala teks di antara tag <p> dan </p> akan menjadi satu paragraf.

Daftar

Daftar bisa bikin informasi lebih rapi dan mudah dibaca. Ada dua tipe daftar: terurut dan tak terurut.

Contoh Daftar Tak Terurut:

<ul>
    <li>Belajar HTML</li>
    <li>Belajar CSS</li>
    <li>Belajar JavaScript</li>
</ul>

Contoh Daftar Terurut:

<ol>
    <li>Langkah pertama</li>
    <li>Langkah kedua</li>
    <li>Langkah ketiga</li>
</ol>

Penjelasan:

<ul> adalah untuk daftar tak terurut (unordered list), sedangkan <ol> untuk daftar terurut (ordered list). Setiap item dalam daftar menggunakan <li> (list item).

Gambar

Gambar bikin website lebih menarik dan visual. Dengan HTML, kamu bisa menampilkan gambar dari URL atau file lokal.

Contoh Gambar:

<img src="gambar-koding.jpg" alt="Gambar Koding" >

Penjelasan:

src: Menunjukkan sumber gambar. Bisa berupa URL atau path ke file gambar di komputermu.

alt: Teks alternatif jika gambarnya tidak bisa ditampilkan. Ini juga penting untuk aksesibilitas.

Menyisipkan Tautan

Tautan adalah jembatan ke halaman lain atau website lain. Di HTML, kita menggunakan tag <a> untuk membuat tautan.

Contoh Tautan:

<a href="https://www.google.com">Kunjungi Google!</a>

Penjelasan:

<a> adalah tag untuk membuat tautan, dan href menunjukkan tujuan dari tautan tersebut, apakah ke halaman lain atau situs web lain.
